MELISA YULIA KRISTINA

A former educator

I’m a former educator who’s dreaming of having a career in tech so I can keep learning new things about technology everyday, be creative and more in control of my job prospects. I have started to learn coding in January 2019 and I feel like I finally found something I really want to do for a living. I’m an enthusiastic learner with adaptability skill and perseverance. I don’t give up easily and I think of difficulties as a chance to build my competence. I started by using free and/or affordable resources and courses to learn HTML, CSS, JavaScript, front-end libraries (Bootstrap, jQuery, ReactJS) and a bit SQL, Git, GitHub and Python. I also tried to learn with another people around the world by doing some remote pair programming and created simple projects with my friends. I think my decision to be a developer is one of the best decisions I made in life because I enjoy the bliss of solving problems and being able to create things with programming.
